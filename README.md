## 📊 项目二：销售数据运营看板分析（excel）

通过阿里云获取天猫月度销售数据，基于订单数据，制作数据看板以及进行分析。

### 核心工具
Excel / 数据透视表 / 漏斗分析

### 关键发现
- **高流量低转化高退货**：通过漏斗模型，发现支付流失率 14%，退款率高达 30.08%。
- **实际收入缩水**：账面销售额 190 万，实际到账仅 133 万。
- **区域洞察**：北京、广东省为销售高地，但客单价偏低，存在优化空间。

### 决策推动
- 设计 A/B 实验验证支付页面优化效果。
- 提出退款专项治理方案（质检升级、详情页优化、物流更换）。
